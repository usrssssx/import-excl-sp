# Release Notes v1.0.0-mvp

Дата: 2026-03-13

## Что сделано

- Реализован Local App для Bitrix24 (`install` + `handler` + iframe-контекст).
- Реализован импорт Excel в смарт-процессы через очередь:
  - чтение файла;
  - валидация строк;
  - пакетная отправка в Bitrix (`batch`, `crm.item.add`);
  - прогресс и итог импорта;
  - выгрузка ошибок в отдельный Excel.
- Реализовано управление доступом:
  - права на СП;
  - доступ для всех/по пользователям/по отделам;
  - отдельный экран админов приложения.
- Усилена валидация импорта:
  - строгие проверки типов;
  - понятные сообщения ошибок;
  - единая нормализация `ufCrm_*`;
  - ограничения файла (размер/расширение/MIME).
- Добавлены автотесты:
  - unit: нормализация UF и валидация строк;
  - feature: загрузка файла, создание `import_job`, прогресс, итог;
  - в feature-тестах Bitrix API замокан (без внешних вызовов).
- Улучшен UX результата:
  - итоговая строка `N успешно / M ошибки / всего`;
  - отдельные кнопки `Загрузить ещё файл` и `Выбрать другой СП`;
  - кнопка скачивания ошибок показывается только при реальном наличии файла.
- Обновлена эксплуатационная документация (`docs/ops.md`).

## Ограничения MVP

- Нет проверки подписи входящих запросов Bitrix24.
- Не реализован расширенный импорт сложных типов (например, `file`).
- Производительность зависит от доступности Bitrix REST и конфигурации воркера очереди.
